CSS Coverage Demo

  1. Open the developer toolbar (Shift+F2)
  2. Type "csscoverage start"
  3. Click through the pages in turn.
    Next: Page 1
  4. When you're done collecting data, close the notification bar or type "csscoverage stop"
  5. In the style editor check for selectors with a strikeout, or type "csscoverage report"
